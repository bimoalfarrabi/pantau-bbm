# Roadmap

## MVP

Target:

Membuat aplikasi publik yang dapat menampilkan harga BBM terbaru dan riwayat perubahan harga.

Fitur:

- Homepage.
- Search daerah.
- Halaman detail daerah.
- Harga terbaru.
- History harga.
- Scheduler sync.
- Database cache.
- Attribution Bensin API.
- Disclaimer bukan website resmi.

## Phase 2: Comparison & Chart

Fitur:

- Perbandingan harga antar daerah.
- Grafik riwayat harga.
- Filter produk lebih lengkap.
- Sort harga termurah/termahal.

## Phase 3: PWA

Fitur:

- Installable web app.
- Offline basic shell.
- Shortcut ke halaman daerah favorit.

## Phase 4: Public API

Fitur:

- Endpoint publik PantauBBM.
- Rate limit.
- Dokumentasi API internal.
- Attribution requirement.

## Phase 5: Notification

Fitur:

- Subscribe daerah.
- Notifikasi perubahan harga.
- Email atau web push.

Catatan:

Fitur ini membutuhkan login atau minimal mekanisme subscription.

## Phase 6: Analytics & Data Export

Fitur:

- Export CSV.
- Statistik perubahan harga.
- Dataset historis.
- Halaman insight.
